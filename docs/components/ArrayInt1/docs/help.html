This component acts as an array for integer data. When the input event [In]write is asserted, the input data [In]data is added to the array at the index specified by [In]index and the output event [Out]write is triggered. When the input event [In]read is asserted, the data in the array at the index specified by [In]index is presented to the output [Out]data and the output event [Out]read is triggered.

Please note that the array is zero indexed and the maximum index is therefore the value of the parameter specifying the maximum size of the array - 1. If an index of less than 0 or greater than or equal to the maximum array size is used when writing to or reading from the array then the corresponding output event [Out]err will be triggered. Also note that data is persistent at the output [Out]data.

The default value parameter specifies the default value for uninitialised values in the array.
